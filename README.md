Professional Clinic Management System

## Features
- **Real-time Dashboard**: Live analytics for patients and financials.
- **Smart EMR**: Electronic Medical Records with 3D tooth marking.
- **Appointment Scheduler**: Optimized booking and clinic management.
- **Financial Analytics**: Tracking revenue, earnings, and payroll.
- **AI-Powered**: profile reconstruction and clinical assistance.

## Tech Stack
- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS + Framer Motion
- **Icons**: Lucide React
- **Backend**: Supabase (PostgreSQL)
- **AI**: Google Gemini Pro

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure environment variables in `.env`.
4. Run locally: `npm run dev`
